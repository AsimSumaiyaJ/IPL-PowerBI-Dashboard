## Project Overview
This project is an end-to-end Power BI dashboard developed to analyze **Indian Premier League (IPL)** data from **2008 to 2025**.  
The dashboard provides interactive insights into team performance, season outcomes, player achievements, and tournament statistics using dynamic filters and visuals.

This project is designed to meet real-world **business and analytical requirements**.

---

##  Business Objective
The main objective of this project is to build an **interactive Power BI dashboard** that enables users to analyze IPL seasons (2008â€“2025) and instantly view season-wise, team-wise, and player-wise insights.

A season slicer allows users to dynamically explore data for any selected IPL season.

---

## Primary KPIs
When a season is selected, the dashboard dynamically displays:

- **Winner Team** of the selected season  
  - Team name  
  - Dynamically rendered team logo  

- **Runner-Up Team** of the selected season  
  - Team name  
  - Dynamically rendered team logo  

---

##  Secondary KPIs
For the selected season, the dashboard provides:

- Total Sixes
- Total Fours
- Total Matches Played
- Total Teams Participated
- Total Centuries
- Total Half-Centuries
- Total Venues Used
